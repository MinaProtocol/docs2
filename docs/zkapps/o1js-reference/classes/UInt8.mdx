### new UInt8(x)

```ts
new UInt8(x): UInt8
```

Create a [UInt8](UInt8.mdx) from a bigint or number.
The max value of a [UInt8](UInt8.mdx) is `2^8 - 1 = 255`.

**Warning**: Cannot overflow past 255, an error is thrown if the result is greater than 255.
