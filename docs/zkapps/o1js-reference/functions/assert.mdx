```ts
assert(stmt: boolean | Bool, message?: string): asserts stmt
```

Assert that a statement is true. If the statement is false, throws an error with the given message.
Can be used in provable code.

## Parameters

• **stmt**: `boolean` \| [`Bool`](../classes/Bool.mdx)

• **message?**: `string`

## Returns

`asserts stmt`
