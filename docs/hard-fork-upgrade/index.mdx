# Hard Fork Upgrade

There is already a good description of hardfork in [Hard Fork process](mina-protocol/hardforkssteps) article

## How to prepare for Berkeley hardfork

Berkeley Hard Fork updgrades all nodes including archives. One of necessary step is to migrate
archive databases from current mainnet format to berkeley. This will unfortunately requires some efforts from
node operators and exchanges. 

Read more about archive data migration:

- [How to prepare for archive migration](/hard-fork-upgrade/prerequisites-and-planning)
- [How to perform archive migration](/hard-fork-upgrade/migrating-mainne-database-to-berkeley)
